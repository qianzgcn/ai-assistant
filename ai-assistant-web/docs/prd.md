# ai-assistant PRD

## 1. 产品目标
ai-assistant 是一个桌面优先的中文 AI 对话 Web 应用，定位为“前端先行的 AI 聊天工作台”。在真实后端尚未接入前，应用通过 mock 服务模拟主流 AI 对话产品的核心体验，包括多会话、流式回复、本地持久化与重新生成。

## 2. 目标用户
- 需要先把 AI 对话前端界面做出来的前端工程师
- 需要在接后端前先验证交互与状态流转的产品或设计同学
- 希望以本地数据快速演示 AI 聊天体验的团队

## 3. 核心能力
- 多会话管理：新建、切换、搜索、时间分组、重命名、删除
- 对话能力：欢迎页、推荐问题、流式回复、停止生成、重新生成
- 内容渲染：Markdown、表格、引用、代码高亮、代码复制
- 本地持久化：使用 IndexedDB 保存会话与消息，刷新页面后仍可恢复
- mock 联调：输入 `[mock-error]` 注入错误态，便于前端联调异常流程

## 4. 核心流程
1. 用户进入应用，左侧看到会话列表，右侧看到欢迎页。
2. 用户点击推荐问题或直接输入消息。
3. 系统自动创建新会话，生成标题并开始流式返回助手消息。
4. 用户可切换到其它会话，再切回当前会话继续查看历史与生成状态。
5. 用户可重命名或删除会话，也可对最近一次助手回复执行重新生成。

## 5. 非目标
- 登录与账号体系
- 云同步与多设备同步
- 文件上传、语音输入、图片理解
- 深色模式
- 历史消息编辑分叉

## 6. 验收标准
- 新建会话发送消息后，可看到流式回复
- 左侧会话切换不丢失历史消息
- 页面刷新后数据仍可恢复
- 标题可自动生成，也可手动修改
- 删除会话后可自动切换到下一个可用会话
- 重新生成仅作用于最近一次助手回复
- Markdown 与代码块渲染正确
